# Textmate bundle for Laravel 4

* Laravel 4 snippets
* Blade language grammar
* Artisan commands
* Helpers snippets
* Blade snippets

![Preview](screenshot.png "Menu preview")

## Installing

	cd ~/Library/Application\ Support/Avian/Bundles/
	git clone https://github.com/loranger/Laravel.tmbundle.git
